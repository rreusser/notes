    Provides error bounds for the solution to a system with a complex
    triangular band matrix using alternative indexing semantics.

    Parameters
    ----------
    uplo: string
        specifies whether the upper or lower triangular part is referenced.

    trans: string
        specifies whether the matrix should be transposed.

    diag: string
        specifies whether the matrix is unit triangular.

    N: integer
        number of columns.

    kd: number
        `kd`.

    nrhs: integer
        number of right-hand sides.

    AB: Float64Array
        `AB`.

    strideAB1: integer
        stride of dimension 1 of `AB`.

    strideAB2: integer
        stride of dimension 2 of `AB`.

    offsetAB: integer
        starting index for `AB`.

    B: Float64Array
        `B`.

    strideB1: integer
        stride of dimension 1 of `B`.

    strideB2: integer
        stride of dimension 2 of `B`.

    offsetB: integer
        starting index for `B`.

    X: Float64Array
        `X`.

    strideX1: integer
        stride of dimension 1 of `X`.

    strideX2: integer
        stride of dimension 2 of `X`.

    offsetX: integer
        starting index for `X`.

    FERR: Float64Array
        `FERR`.

    strideFERR: integer
        stride length for `FERR`.

    offsetFERR: integer
        starting index for `FERR`.

    BERR: Float64Array
        `BERR`.

    strideBERR: integer
        stride length for `BERR`.

    offsetBERR: integer
        starting index for `BERR`.

    WORK: Float64Array
        `WORK`.

    strideWORK: integer
        stride length for `WORK`.

    offsetWORK: integer
        starting index for `WORK`.

    RWORK: Float64Array
        `RWORK`.

    strideRWORK: integer
        stride length for `RWORK`.

    offsetRWORK: integer
        starting index for `RWORK`.
