    Solve A*X=B with complex symmetric A using factorization from zsytrf_rook

    Parameters
    ----------
        order: string
            Row-major (C-style) or column-major (Fortran-style) order.

        uplo: string
            specifies the operation type.

        N: integer
            number of columns.

        nrhs: integer
            nrhs.

        A: Float64Array
            input matrix.

        LDA: integer
            leading dimension of `A`.

        IPIV: Int32Array
            input array.

        strideIPIV: integer
            stride length for `IPIV`.

        offsetIPIV: integer
            starting index for `IPIV`.

        B: Float64Array
            output matrix.

        LDB: integer
            leading dimension of `B`.
