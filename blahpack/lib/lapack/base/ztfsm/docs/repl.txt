{{alias}}( transr, side, uplo, trans, diag, M, N, alpha, A, B )
    Solves a matrix equation with a complex triangular matrix in Rectangular
    Full Packed format.

    Parameters
    ----------
    transr: string
        `transr`.

    side: string
        specifies the side of the operation.

    uplo: string
        specifies whether the upper or lower triangular part is referenced.

    trans: string
        specifies whether the matrix should be transposed.

    diag: string
        specifies whether the matrix is unit triangular.

    M: integer
        number of rows.

    N: integer
        number of columns.

    alpha: number
        scalar constant.

    A: Float64Array
        `A`.

    B: Float64Array
        `B`.
