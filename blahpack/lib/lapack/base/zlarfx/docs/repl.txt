
    zlarfx.ndarray( side, M, N, v, strideV, offsetV, tau, C, strideC1, strideC2, offsetC, WORK, strideWORK, offsetWORK )
        Applies an elementary reflector to a general rectangular matrix with loop unrolling when the reflector has order at most 10.

        Parameters
        ----------
        side: string
            specifies the operation type.

        M: NonNegativeInteger
            number of rows.

        N: NonNegativeInteger
            number of columns.

        v: Float64Array
            input array.

        strideV: integer
            stride length for `v`.

        offsetV: NonNegativeInteger
            starting index for `v`.

        tau: Complex128
            tau.

        C: Float64Array
            input matrix.

        strideC1: integer
            stride of the first dimension of `C`.

        strideC2: integer
            stride of the second dimension of `C`.

        offsetC: NonNegativeInteger
            starting index for `C`.

        WORK: Float64Array
            output array.

        strideWORK: integer
            stride length for `WORK`.

        offsetWORK: NonNegativeInteger
            starting index for `WORK`.
