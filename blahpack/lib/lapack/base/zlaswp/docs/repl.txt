
    zlaswp.ndarray( N, A, strideA1, strideA2, offsetA, k1, k2, IPIV, strideIPIV, offsetIPIV, incx )
        Performs a series of row interchanges on a complex double-precision matrix.

        Parameters
        ----------
        N: integer
            Number of columns in A.
        A: Complex128Array
            Input matrix.
        strideA1: integer
            Stride of the first dimension of A.
        strideA2: integer
            Stride of the second dimension of A.
        offsetA: integer
            Index offset for A (in complex elements).
        k1: integer
            Index of first row to interchange (0-based).
        k2: integer
            Index of last row to interchange (0-based).
        IPIV: Int32Array
            Vector of pivot indices (0-based).
        strideIPIV: integer
            Stride for IPIV.
        offsetIPIV: integer
            Index offset for IPIV.
        incx: integer
            Direction in which to apply pivots.

        Returns
        -------
        out: Complex128Array
            Permuted matrix A.
