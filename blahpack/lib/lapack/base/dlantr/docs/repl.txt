        `norm`.

    uplo: string

    diag: string
        specifies whether the matrix is unit triangular.

    M: integer

    N: integer
        number of columns.

    A: Float64Array
        `A`.

    strideA1: integer
        stride of dimension 1 of `A`.

    strideA2: integer
        stride of dimension 2 of `A`.

    offsetA: integer
        starting index for `A`.

    WORK: Float64Array

    strideWORK: integer
