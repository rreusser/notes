{{alias}}( order, trans, N, nrhs, A, LDA, IPIV, strideIPIV, B, LDB )
    Solves a system of linear equations.

    Parameters
    ----------
    order: string
        Row-major (C-style) or column-major (Fortran-style) order.

    trans: string
        specifies whether the matrix should be transposed.

    N: integer
        number of columns.

    nrhs: integer
        number of right-hand sides.

    A: Float64Array
        `A`.

    LDA: integer
        leading dimension of `A`.

    IPIV: Int32Array
        `IPIV`.

    strideIPIV: integer
        stride length for `IPIV`.

    B: Float64Array
        `B`.

    LDB: integer
        leading dimension of `B`.

    Returns
    -------
    out: Float64Array
        Result.
