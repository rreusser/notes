{{alias}}( order, uplo, trans, diag, normin, N, A, LDA, x, strideX, scale, CNORM, strideCNORM )
    Solves a triangular system with scaling to prevent overflow.

    Parameters
    ----------
    order: string
        Row-major (C-style) or column-major (Fortran-style) order.

    uplo: string
        specifies whether the upper or lower triangular part is referenced.

    trans: string
        specifies whether the matrix should be transposed.

    diag: string
        specifies whether the matrix is unit triangular.

    normin: string
        `normin`.

    N: integer
        number of columns.

    A: Float64Array
        `A`.

    LDA: integer
        leading dimension of `A`.

    x: Float64Array
        `x`.

    strideX: integer
        stride length for `X`.

    scale: number
        `scale`.

    CNORM: Float64Array
        `CNORM`.

    strideCNORM: integer
        stride length for `CNORM`.

    Returns
    -------
    out: Float64Array
        Result.
