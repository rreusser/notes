{{alias}}( order, jobz, uplo, N, A, LDA, w, strideW, WORK, strideWORK, lwork, RWORK, strideRWORK )
    Computes all eigenvalues and, optionally, eigenvectors of a complex
    Hermitian.

    Parameters
    ----------
    order: string
        Row-major (C-style) or column-major (Fortran-style) order.

    jobz: string
        `jobz`.

    uplo: string
        specifies whether the upper or lower triangular part is referenced.

    N: integer
        number of columns.

    A: Float64Array
        `A`.

    LDA: integer
        leading dimension of `A`.

    w: Float64Array
        `w`.

    strideW: integer
        stride length for `W`.

    WORK: Float64Array
        `WORK`.

    strideWORK: integer
        stride length for `WORK`.

    lwork: integer
        `lwork`.

    RWORK: Float64Array
        `RWORK`.

    strideRWORK: integer
        stride length for `RWORK`.

    Returns
    -------
    out: Float64Array
