    Parameters
    ----------
        order: string
            Row-major (C-style) or column-major (Fortran-style) order.

        side: string
            specifies the operation type.

        eigsrc: string
            specifies the operation type.

        initv: string
            specifies the operation type.

        SELECT: Float64Array
            input array.

        strideSELECT: integer
            stride length for `SELECT`.

        N: integer
            number of columns.

        H: Float64Array
            input matrix.

        LDH: integer
            leading dimension of `H`.

        w: Float64Array
            input array.

        strideW: integer
            stride length for `w`.

        VL: Float64Array
            input matrix.

        LDVL: integer
            leading dimension of `VL`.

        VR: Float64Array
            input matrix.

        LDVR: integer
            leading dimension of `VR`.

        mm: integer
            mm.

        M: integer
            number of rows.

        WORK: Float64Array
            input array.

        strideWORK: integer
            stride length for `WORK`.

        RWORK: Float64Array
            input array.

        strideRWORK: integer
            stride length for `RWORK`.

            input array.

        strideIFAILL: integer

            output array.
