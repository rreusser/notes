
    dgetf2.ndarray( M, N, A, strideA1, strideA2, offsetA, IPIV, strideIPIV, offsetIPIV )

        Parameters
        ----------
        M: NonNegativeInteger
            number of rows.

        N: NonNegativeInteger
            number of columns.

        A: Float64Array
            input matrix.

        strideA1: integer
            stride of the first dimension of `A`.

        strideA2: integer
            stride of the second dimension of `A`.

        offsetA: NonNegativeInteger
            starting index for `A`.

        IPIV: Int32Array
            output array.

        strideIPIV: integer
            stride length for `IPIV`.

        offsetIPIV: NonNegativeInteger
