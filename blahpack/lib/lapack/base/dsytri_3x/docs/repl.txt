
{{alias}}( order, uplo, N, A, LDA, e, strideE, IPIV, strideIPIV, offsetIPIV, WORK, strideWORK, nb )
    Compute the inverse of a real symmetric indefinite matrix using factorization computed by dsytrf_rk (worker routine called by dsytri_3)

    Parameters
    ----------
        order: string
            Row-major (C-style) or column-major (Fortran-style) order.

        uplo: string
            specifies the operation type.

        N: integer
            number of columns.

        A: Float64Array
            input matrix.

        LDA: integer
            leading dimension of `A`.

        e: Float64Array
            input array.

        strideE: integer
            stride length for `e`.

        IPIV: Int32Array
            input array.

        strideIPIV: integer
            stride length for `IPIV`.

        offsetIPIV: integer
            starting index for `IPIV`.

        WORK: Float64Array
            output array.

        strideWORK: integer
            stride length for `WORK`.

        nb: integer
            nb.

    Returns
    -------
    out: integer
        Result.
