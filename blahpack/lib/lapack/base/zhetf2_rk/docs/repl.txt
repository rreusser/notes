
{{alias}}( order, uplo, N, A, LDA, e, strideE, IPIV, strideIPIV, offsetIPIV )
    Factorizes a Hermitian matrix A using the bounded Bunch-Kaufman (rook) diagonal pivoting method (_rk format).

    Parameters
    ----------
        order: string
            Row-major (C-style) or column-major (Fortran-style) order.

        uplo: string
            specifies the operation type.

        N: integer
            number of columns.

        A: Float64Array
            input matrix.

        LDA: integer
            leading dimension of `A`.

        e: Float64Array
            input array.

        strideE: integer
            stride length for `e`.

        IPIV: Int32Array
            output array.

        strideIPIV: integer
            stride length for `IPIV`.

        offsetIPIV: integer
            starting index for `IPIV`.

    Returns
    -------
    out: integer
        Result.
