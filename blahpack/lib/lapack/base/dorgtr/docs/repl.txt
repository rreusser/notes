{{alias}}( order, uplo, N, A, LDA, TAU, strideTAU, WORK, strideWORK )
    Generates an orthogonal matrix Q which is defined as the product of N-1.

    Parameters
    ----------
    order: string
        Row-major (C-style) or column-major (Fortran-style) order.

    uplo: string
        specifies whether the upper or lower triangular part is referenced.

    N: integer
        number of columns.

    A: Float64Array
        `A`.

    LDA: integer
        leading dimension of `A`.

    TAU: Float64Array
        `TAU`.

    strideTAU: integer
        stride length for `TAU`.

    WORK: Float64Array
        `WORK`.

    strideWORK: integer
        stride length for `WORK`.

    Returns
    -------
    out: Float64Array
