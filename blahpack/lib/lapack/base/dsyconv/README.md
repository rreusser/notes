> Converts a symmetric matrix factored by dsytrf to standard L*D*L^T form and vice versa

## Usage

```javascript
var dsyconv = require( '@stdlib/lapack/base/dsyconv' );
```

#### dsyconv.ndarray( uplo, way, N, A, strideA1, strideA2, offsetA, IPIV, strideIPIV, offsetIPIV, e, strideE, offsetE )

Converts a symmetric matrix factored by dsytrf to standard L*D*L^T form and vice versa

The function has the following parameters:

-   **uplo**: specifies the operation type.
-   **way**: specifies the operation type.
-   **N**: number of columns.
-   **A**: input matrix.
-   **strideA1**: stride of the first dimension of `A`.
-   **strideA2**: stride of the second dimension of `A`.
-   **offsetA**: starting index for `A`.
-   **IPIV**: input array.
-   **strideIPIV**: stride length for `IPIV`.
-   **offsetIPIV**: starting index for `IPIV`.
-   **e**: output array.
-   **strideE**: stride length for `e`.
-   **offsetE**: starting index for `e`.
