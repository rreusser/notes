
{{alias}}( order, uplo, N, nb, kb, A, LDA, e, strideE, IPIV, strideIPIV, offsetIPIV, W, LDW )
    DLASYF_RK computes a partial factorization of a real symmetric indefinite matrix using bounded Bunch-Kaufman (rook) diagonal pivoting method, producing _rk format output

    Parameters
    ----------
        order: string
            Row-major (C-style) or column-major (Fortran-style) order.

        uplo: string
            specifies the operation type.

        N: integer
            number of columns.

        nb: integer
            nb.

        kb: integer
            kb.

        A: Float64Array
            input matrix.

        LDA: integer
            leading dimension of `A`.

        e: Float64Array
            input array.

        strideE: integer
            stride length for `e`.

        IPIV: Int32Array
            input array.

        strideIPIV: integer
            stride length for `IPIV`.

        offsetIPIV: integer
            starting index for `IPIV`.

        W: Float64Array
            output matrix.

        LDW: integer
            leading dimension of `W`.

    Returns
    -------
    out: integer
        Result.
