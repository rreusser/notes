The function has the following parameters:

-   **strideX**: stride length for `x`.
-   **offsetX**: starting index for `x`.
-   **y**: input array.
-   **strideY**: stride length for `y`.
-   **offsetY**: starting index for `y`.
-   **c**: output array.
-   **strideC**: stride length for `c`.
