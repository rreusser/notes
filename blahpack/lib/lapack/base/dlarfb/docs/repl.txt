    Parameters
    ----------
    order: string

    side: string
        specifies the side of the operation.

    trans: string
        specifies whether the matrix should be transposed.

    direct: string
        `direct`.

    storev: string
        `storev`.

    M: integer
        number of rows.

    N: integer
        number of columns.

    K: integer
        inner dimension.

    V: Float64Array

    LDV: integer
        leading dimension of `V`.

    T: Float64Array

    LDT: integer
        leading dimension of `T`.

    C: Float64Array
        `C`.

    LDC: integer
        leading dimension of `C`.

    WORK: Float64Array
        `WORK`.
