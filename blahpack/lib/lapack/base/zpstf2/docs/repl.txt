{{alias}}( order, uplo, N, A, LDA, PIV, RANK, tol, WORK )
    Computes the Cholesky factorization with complete pivoting of a complex
    Hermitian positive semi-definite matrix (unblocked algorithm).

    Parameters
    ----------
    order: string
        Row-major (C-style) or column-major (Fortran-style) order.

    uplo: string
        specifies whether the upper or lower triangular part is referenced.

    N: integer
        number of columns.

    A: Float64Array
        `A`.

    LDA: integer
        leading dimension of `A`.

    PIV: Float64Array
        `PIV`.

    RANK: Float64Array
        `RANK`.

    tol: number
        `tol`.

    WORK: Float64Array
        `WORK`.
