        Parameters
        ----------
        uplo: string
            specifies the operation type.

        N: NonNegativeInteger
            number of columns.

        nrhs: integer
            nrhs.

        AP: Float64Array
            input array.

        strideAP: integer
            stride length for `AP`.

        offsetAP: NonNegativeInteger
            starting index for `AP`.

        B: Float64Array
            output matrix.

        strideB1: integer
            stride of the first dimension of `B`.

        strideB2: integer
            stride of the second dimension of `B`.
