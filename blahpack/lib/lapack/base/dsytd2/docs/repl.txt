{{alias}}( order, uplo, N, A, LDA, d, strideD, e, strideE, TAU, strideTAU )
    Reduces a real symmetric matrix A to real symmetric tridiagonal form T.

    Parameters
    ----------
    order: string
        Row-major (C-style) or column-major (Fortran-style) order.

    uplo: string
        specifies whether the upper or lower triangular part is referenced.

    N: integer
        number of columns.

    A: Float64Array
        `A`.

    LDA: integer
        leading dimension of `A`.

    d: Float64Array
        `d`.

    strideD: integer
        stride length for `D`.

    e: Float64Array
        `e`.

    strideE: integer
        stride length for `E`.

    TAU: Float64Array
        `TAU`.

    strideTAU: integer
        stride length for `TAU`.

    Returns
    -------
    out: Float64Array
        Result.
