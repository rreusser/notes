{{alias}}( order, uplo, N, AP, A, LDA )
    Copies a complex triangular matrix from standard packed format (TP) to
    full format (TR).

    Parameters
    ----------
    order: string
        Row-major (C-style) or column-major (Fortran-style) order.

    uplo: string
        specifies whether the upper or lower triangular part is referenced.

    N: integer
        number of columns.

    AP: Float64Array
        `AP`.

    A: Float64Array
        `A`.

    LDA: integer
        leading dimension of `A`.

    Returns
    -------
    out: Float64Array
        Result.
