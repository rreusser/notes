
{{alias}}( order, uplo, N, nrhs, A, LDA, e, strideE, IPIV, strideIPIV, offsetIPIV, B, LDB )
    Solve A*X = B for symmetric (non-Hermitian) complex A using factorization computed by ZSYTRF_RK

    Parameters
    ----------
        order: string
            Row-major (C-style) or column-major (Fortran-style) order.

        uplo: string
            specifies the operation type.

        N: integer
            number of columns.

        nrhs: integer
            nrhs.

        A: Float64Array
            input matrix.

        LDA: integer
            leading dimension of `A`.

        e: Float64Array
            input array.

        strideE: integer
            stride length for `e`.

        IPIV: Int32Array
            input array.

        strideIPIV: integer
            stride length for `IPIV`.

        offsetIPIV: integer
            starting index for `IPIV`.

        B: Float64Array
            output matrix.

        LDB: integer
            leading dimension of `B`.

    Returns
    -------
    out: integer
        Result.
