
    zgetrf.ndarray( M, N, A, strideA1, strideA2, offsetA, IPIV, strideIPIV, offsetIPIV )
        Computes an LU factorization of a general complex M-by-N matrix using
        partial pivoting with row interchanges.

        Parameters
        ----------
        M: NonNegativeInteger
            number of rows.

        N: NonNegativeInteger
            number of columns.

        A: Complex128Array
            input complex matrix.

        strideA1: integer
            stride of the first dimension of `A` (complex elements).

        strideA2: integer
            stride of the second dimension of `A` (complex elements).

        offsetA: NonNegativeInteger
            starting index for `A` (complex elements).

        IPIV: Int32Array
            output array.

        strideIPIV: integer
            stride length for `IPIV`.

        offsetIPIV: NonNegativeInteger
            starting index for `IPIV`.
