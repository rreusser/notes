
    dpteqr.ndarray( compz, N, d, strideD, offsetD, e, strideE, offsetE, Z, strideZ1, strideZ2, offsetZ, WORK, strideWORK, offsetWORK )

        Parameters
        ----------
        compz: string
            specifies the operation type.

        N: NonNegativeInteger
            number of columns.

        d: Float64Array
            input array.

        strideD: integer
            stride length for `d`.

        offsetD: NonNegativeInteger
            starting index for `d`.

        e: Float64Array
            input array.

        strideE: integer
            stride length for `e`.

        offsetE: NonNegativeInteger
            starting index for `e`.

        Z: Float64Array
            input matrix.

        strideZ1: integer
            stride of the first dimension of `Z`.

        strideZ2: integer
            stride of the second dimension of `Z`.

        offsetZ: NonNegativeInteger
            starting index for `Z`.

        WORK: Float64Array
            output array.

        strideWORK: integer
            stride length for `WORK`.

        offsetWORK: NonNegativeInteger
            starting index for `WORK`.
