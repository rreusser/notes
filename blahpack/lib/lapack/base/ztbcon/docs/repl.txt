    Estimates the reciprocal condition number of a complex triangular band
    matrix.

    Parameters
    ----------
    norm: string
        `norm`.

    uplo: string
        specifies whether the upper or lower triangular part is referenced.

    diag: string
        specifies whether the matrix is unit triangular.

        number of columns.

    kd: number
        `kd`.

    AB: Float64Array
        `AB`.

    LDAB: integer
        leading dimension of `AB`.

    RCOND: number

    WORK: Float64Array
        `WORK`.

    strideWORK: integer
        stride length for `WORK`.

    RWORK: Float64Array
