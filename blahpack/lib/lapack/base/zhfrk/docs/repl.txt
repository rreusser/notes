{{alias}}( transr, uplo, trans, N, K, alpha, A, LDA, beta, C )
    Performs a Hermitian rank-k operation for a matrix in Rectangular Full
    Packed format.

    Parameters
    ----------
    transr: string
        `transr`.

    uplo: string
        specifies whether the upper or lower triangular part is referenced.

    trans: string
        specifies whether the matrix should be transposed.

    N: integer
        number of columns.

    K: integer
        inner dimension.

    alpha: number
        scalar constant.

    A: Float64Array
        `A`.

    LDA: integer
        leading dimension of `A`.

    beta: number
        scalar constant.

    C: Float64Array
        `C`.
