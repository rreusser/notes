{{alias}}( order, uplo, trans, diag, N, K, A, LDA, x, strideX )
    Solves one of the systems of equations:.

    Parameters
    ----------
    order: string
        Row-major (C-style) or column-major (Fortran-style) order.

    uplo: string
        specifies whether the upper or lower triangular part is referenced.

    trans: string
        specifies whether the matrix should be transposed.

    diag: string
        specifies whether the matrix is unit triangular.

    N: integer
        number of columns.

    K: integer
        inner dimension.

    A: Float64Array
        `A`.

    LDA: integer
        leading dimension of `A`.

    x: Float64Array
        `x`.

    strideX: integer
        stride length for `X`.
