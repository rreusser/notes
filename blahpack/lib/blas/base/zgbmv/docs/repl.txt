
    zgbmv.ndarray( trans, M, N, kl, ku, alpha, A, strideA1, strideA2, offsetA, x, strideX, offsetX, beta, y, strideY, offsetY )
        Perform one of the matrix-vector operations y := alpha*A*x + beta*y or y := alpha*A**T*x + beta*y or y := alpha*A**H*x + beta*y where A is a general band matrix.

        Parameters
        ----------
        trans: string
            specifies the operation type.

        M: NonNegativeInteger
            number of rows.

        N: NonNegativeInteger
            number of columns.

        kl: integer
            kl.

        ku: integer
            ku.

        alpha: Complex128
            scalar constant.

        A: Float64Array
            input matrix.

        strideA1: integer
            stride of the first dimension of `A`.

        strideA2: integer
            stride of the second dimension of `A`.

        offsetA: NonNegativeInteger
            starting index for `A`.

        x: Float64Array
            input array.

        strideX: integer
            stride length for `x`.

        offsetX: NonNegativeInteger
            starting index for `x`.

        beta: Complex128
            scalar constant.

        y: Float64Array
            output array.

        strideY: integer
            stride length for `y`.

        offsetY: NonNegativeInteger
            starting index for `y`.
