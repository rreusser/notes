{{alias}}( order, uplo, trans, N, K, alpha, A, LDA, beta, C, LDC )
    Performs one of the Hermitian rank-k operations:.

    Parameters
    ----------
    order: string
        Row-major (C-style) or column-major (Fortran-style) order.

    uplo: string
        specifies whether the upper or lower triangular part is referenced.

    trans: string
        specifies whether the matrix should be transposed.

    N: integer
        number of columns.

    K: integer
        inner dimension.

    alpha: number
        scalar constant.

    A: Float64Array
        `A`.

    LDA: integer
        leading dimension of `A`.

    beta: number
        scalar constant.

    C: Float64Array
        `C`.

    LDC: integer
        leading dimension of `C`.

    Returns
    -------
    out: Float64Array
        Result.
