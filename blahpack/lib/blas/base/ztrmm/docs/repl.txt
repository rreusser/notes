{{alias}}( order, side, uplo, transa, diag, M, N, alpha, A, LDA, B, LDB )
    Perform one of the matrix-matrix operations.

    Parameters
    ----------
    order: string
        Row-major (C-style) or column-major (Fortran-style) order.

    side: string
        specifies the side of the operation.

    uplo: string
        specifies whether the upper or lower triangular part is referenced.

    transa: string
        specifies the operation for matrix `A`.

    diag: string
        specifies whether the matrix is unit triangular.

    M: integer
        number of rows.

    N: integer
        number of columns.

    alpha: number
        scalar constant.

    A: Float64Array
        `A`.

    LDA: integer
        leading dimension of `A`.

    B: Float64Array
        `B`.

    LDB: integer
        leading dimension of `B`.

    Returns
    -------
    out: Float64Array
        Result.
