
    dtrmm.ndarray( side, uplo, transa, diag, M, N, alpha, A, strideA1, strideA2, offsetA, B, strideB1, strideB2, offsetB )

        Parameters
        ----------
        side: string
            specifies the operation type.

        uplo: string
            specifies the operation type.

        transa: string
            specifies the operation type.

        diag: string
            specifies the operation type.

        M: NonNegativeInteger
            number of rows.

        N: NonNegativeInteger
            number of columns.

        alpha: number
            scalar constant.

        A: Float64Array
            input matrix.

        strideA1: integer
            stride of the first dimension of `A`.

        strideA2: integer
            stride of the second dimension of `A`.

        offsetA: NonNegativeInteger
            starting index for `A`.

        B: Float64Array
            output matrix.

        strideB1: integer
            stride of the first dimension of `B`.

        strideB2: integer
            stride of the second dimension of `B`.

        offsetB: NonNegativeInteger
