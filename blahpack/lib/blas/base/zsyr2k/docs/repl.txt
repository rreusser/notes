{{alias}}( order, uplo, trans, N, K, alpha, A, LDA, B, LDB, beta, C, LDC )
    Performs one of the symmetric rank-2k operations `C := alpha*A*B**T +
    alpha*B*A**T + beta*C` or `C := alpha*A**T*B + alpha*B**T*A + beta*C`
    where alpha and beta are complex scalars, C is an N-by-N symmetric
    matrix, and A and B are N-by-K or K-by-N matrices.

    Parameters
    ----------
    order: string
        Row-major (C-style) or column-major (Fortran-style) order.

    uplo: string
        specifies whether the upper or lower triangular part is referenced.

    trans: string
        specifies whether the matrix should be transposed.

    N: integer
        number of columns.

    K: integer
        inner dimension.

    alpha: number
        scalar constant.

    A: Float64Array
        `A`.

    LDA: integer
        leading dimension of `A`.

    B: Float64Array
        `B`.

    LDB: integer
        leading dimension of `B`.

    beta: number
        scalar constant.

    C: Float64Array
        `C`.

    LDC: integer
        leading dimension of `C`.

    Returns
    -------
    out: Float64Array
        Result.
