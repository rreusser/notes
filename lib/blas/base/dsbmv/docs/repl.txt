{{alias}}( order, uplo, N, K, alpha, A, LDA, x, strideX, beta, y, strideY )
    Performs the matrix-vector operation `y := alpha*A*x + beta*y`.

    Parameters
    ----------
    order: string
        Row-major (C-style) or column-major (Fortran-style) order.

    uplo: string
        specifies whether the upper or lower triangular part is referenced.

    N: integer
        number of columns.

    K: integer
        inner dimension.

    alpha: number
        scalar constant.

    A: Float64Array
        `A`.

    LDA: integer
        leading dimension of `A`.

    x: Float64Array
        `x`.

    strideX: integer
        stride length for `X`.

    beta: number
        scalar constant.

    y: Float64Array
        `y`.

    strideY: integer
        stride length for `Y`.

    Returns
    -------
    out: Float64Array
        Result.
