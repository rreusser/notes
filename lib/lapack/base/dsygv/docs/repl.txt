{{alias}}( itype, jobz, uplo, N, A, LDA, B, LDB, w, strideW, WORK, strideWORK, lwork )
    Computes all the eigenvalues, and optionally, the eigenvectors of a
    real.

    Parameters
    ----------
    itype: integer
        `itype`.

    jobz: string
        `jobz`.

    uplo: string
        specifies whether the upper or lower triangular part is referenced.

    N: integer
        number of columns.

    A: Float64Array

    LDA: integer
        leading dimension of `A`.

    B: Float64Array
        `B`.

    LDB: integer
        leading dimension of `B`.

    w: Float64Array
        `w`.

    strideW: integer
        stride length for `W`.

    WORK: Float64Array
        `WORK`.

    strideWORK: integer
        stride length for `WORK`.

        `lwork`.
