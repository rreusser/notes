
{{alias}}( order, uplo, N, A, LDA, IPIV, strideIPIV, offsetIPIV )
    Bounded Bunch-Kaufman (rook) factorization of a complex Hermitian indefinite matrix

    Parameters
    ----------
        order: string
            Row-major (C-style) or column-major (Fortran-style) order.

        uplo: string
            specifies the operation type.

        N: integer
            number of columns.

        A: Float64Array
            input matrix.

        LDA: integer
            leading dimension of `A`.

        IPIV: Int32Array
            output array.

        strideIPIV: integer
            stride length for `IPIV`.

        offsetIPIV: integer
            starting index for `IPIV`.
