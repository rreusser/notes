{{alias}}( norm, uplo, diag, N, A, LDA, RCOND, WORK, strideWORK, IWORK, strideIWORK )
    Estimates the reciprocal of the condition number of a triangular matrix
    A.

    Parameters
    ----------
    norm: string
        `norm`.

    uplo: string
        specifies whether the upper or lower triangular part is referenced.

    diag: string
        specifies whether the matrix is unit triangular.

    N: integer
        number of columns.

    A: Float64Array
        `A`.

    LDA: integer
        leading dimension of `A`.

    RCOND: number
        `RCOND`.

    WORK: Float64Array
        `WORK`.

    strideWORK: integer
        stride length for `WORK`.

    IWORK: Int32Array
        `IWORK`.

    strideIWORK: integer
        stride length for `IWORK`.

    Returns
    -------
    out: Float64Array
        Result.
