    where A is a complex tridiagonal matrix.

    Parameters
    ----------

        specifies whether the matrix should be transposed.

    N: integer
        number of columns.

    nrhs: integer
        number of right-hand sides.

    alpha: number
        scalar constant.

    DL: Float64Array
        `DL`.

    strideDL: integer
        stride length for `DL`.

    d: Float64Array

    strideD: integer

    strideDU: integer

    X: Float64Array

    LDX: integer

    beta: number

    B: Float64Array
        `B`.
