{{alias}}( order, side, pivot, direct, M, N, c, strideC, s, strideS, A, LDA )
    Applies a sequence of real plane rotations to a real general rectangular
    matrix.

    Parameters
    ----------
    order: string
        Row-major (C-style) or column-major (Fortran-style) order.

    side: string
        specifies the side of the operation.

    pivot: number
        `pivot`.

    direct: string
        `direct`.

    M: integer
        number of rows.

    N: integer
        number of columns.

    c: Float64Array
        `c`.

    strideC: integer
        stride length for `C`.

    s: Float64Array
        `s`.

    strideS: integer
        stride length for `S`.

    A: Float64Array
        `A`.

    LDA: integer
        leading dimension of `A`.

    Returns
    -------
    out: Float64Array
        Result.
