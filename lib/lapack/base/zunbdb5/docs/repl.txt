
{{alias}}( order, m1, m2, N, X1, strideX1, X2, strideX2, Q1, LDQ1, Q2, LDQ2, WORK, strideWORK )
    Orthogonalizes the complex column vector `X = [X1; X2]` against the columns
    of `Q = [Q1; Q2]`, where the columns of Q are assumed to be orthonormal.
    If the projection collapses to zero, returns a deterministic vector from
    the orthogonal complement instead.

    Parameters
    ----------
        order: string
            Row-major (C-style) or column-major (Fortran-style) order.

        m1: integer
            Dimension of `X1` and number of rows in `Q1`.

        m2: integer
            Dimension of `X2` and number of rows in `Q2`.

        N: integer
            Number of columns in `Q1` and `Q2`.

        X1: Complex128Array
            Top part of the input vector.

        strideX1: integer
            Stride length for `X1` (in complex elements).

        X2: Complex128Array
            Bottom part of the input vector.

        strideX2: integer
            Stride length for `X2` (in complex elements).

        Q1: Complex128Array
            Top part of the orthonormal basis matrix.

        LDQ1: integer
            Leading dimension of `Q1`.

        Q2: Complex128Array
            Bottom part of the orthonormal basis matrix.

        LDQ2: integer
            Leading dimension of `Q2`.

        WORK: Complex128Array
            Workspace array of length at least `N`.

        strideWORK: integer
            Stride length for `WORK` (in complex elements).

    Returns
    -------
    info: integer
        Status code (0 = success).

    Examples
    --------
    > var Q1 = new {{alias:@stdlib/array/complex128}}( [ 1.0, 0.0, 0.0, 0.0 ] );
    > var Q2 = new {{alias:@stdlib/array/complex128}}( [ 0.0, 0.0, 0.0, 0.0 ] );
    > var X1 = new {{alias:@stdlib/array/complex128}}( [ 3.0, 0.0, 4.0, 0.0 ] );
    > var X2 = new {{alias:@stdlib/array/complex128}}( [ 5.0, 0.0, 6.0, 0.0 ] );
    > var WORK = new {{alias:@stdlib/array/complex128}}( 1 );
    > {{alias}}( 'column-major', 2, 2, 1, X1, 1, X2, 1, Q1, 2, Q2, 2, WORK, 1 )
