
{{alias}}( order, uplo, N, A, LDA, e, strideE, IPIV, strideIPIV )
    Computes the bounded Bunch-Kaufman (rook) factorization of a real symmetric
    indefinite matrix in `_rk` storage format (blocked algorithm).

    Parameters
    ----------
        order: string
            Row-major (C-style) or column-major (Fortran-style) order.

        uplo: string
            Triangle of `A` referenced (`'upper'` or `'lower'`).

        N: integer
            Order of the matrix `A`.

        A: Float64Array
            Input/output symmetric matrix. On exit, contains the factorization.

        LDA: integer
            Leading dimension of `A`.

        e: Float64Array
            Output vector containing the off-diagonal entries of `D`.

        strideE: integer
            Stride length for `e`.

        IPIV: Int32Array
            Output pivot index array (0-based; 2x2 pivots use the bitwise NOT
            convention).

        strideIPIV: integer
            Stride length for `IPIV`.

    Returns
    -------
    out: integer
        `0` on success; `k>0` (1-based) if `D(k,k)` is exactly zero.

    Examples
    --------
    > var A = new {{alias:@stdlib/array/float64}}( [ 4, 2, 1, 0, 0, 5, 2, 1, 0, 0, 6, 3, 0, 0, 0, 8 ] );
    > var e = new {{alias:@stdlib/array/float64}}( 4 );
    > var IPIV = new {{alias:@stdlib/array/int32}}( 4 );
    > {{alias}}( 'column-major', 'lower', 4, A, 4, e, 1, IPIV, 1 );


{{alias}}.ndarray( uplo, N, A, strideA1, strideA2, offsetA, e, strideE, offsetE, IPIV, strideIPIV, offsetIPIV )
    Same operation as the BLAS-style API, with full ndarray (stride/offset)
    semantics for every array argument.

    Parameters
    ----------
        uplo: string
            Triangle of `A` referenced (`'upper'` or `'lower'`).

        N: integer
            Order of the matrix `A`.

        A: Float64Array
            Input/output symmetric matrix.

        strideA1: integer
            Stride of dimension 1 of `A`.

        strideA2: integer
            Stride of dimension 2 of `A`.

        offsetA: integer
            Starting index for `A`.

        e: Float64Array
            Output vector containing the off-diagonal entries of `D`.

        strideE: integer
            Stride length for `e`.

        offsetE: integer
            Starting index for `e`.

        IPIV: Int32Array
            Output pivot index array.

        strideIPIV: integer
            Stride length for `IPIV`.

        offsetIPIV: integer
            Starting index for `IPIV`.

    Returns
    -------
    out: integer
        `0` on success; `k>0` (1-based) if `D(k,k)` is exactly zero.

    Examples
    --------
    > var A = new {{alias:@stdlib/array/float64}}( [ 4, 2, 1, 0, 0, 5, 2, 1, 0, 0, 6, 3, 0, 0, 0, 8 ] );
    > var e = new {{alias:@stdlib/array/float64}}( 4 );
    > var IPIV = new {{alias:@stdlib/array/int32}}( 4 );
    > {{alias}}.ndarray( 'lower', 4, A, 1, 4, 0, e, 1, 0, IPIV, 1, 0 );
