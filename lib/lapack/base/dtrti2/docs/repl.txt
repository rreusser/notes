{{alias}}( order, uplo, diag, N, A, LDA )
    Computes the inverse of a real upper or lower triangular matrix.

    Parameters
    ----------
    order: string
        Row-major (C-style) or column-major (Fortran-style) order.

    uplo: string
        specifies whether the upper or lower triangular part is referenced.

    diag: string
        specifies whether the matrix is unit triangular.

    N: integer
        number of columns.

    A: Float64Array
        `A`.

    LDA: integer
        leading dimension of `A`.

    Returns
    -------
    out: Float64Array
        Result.
