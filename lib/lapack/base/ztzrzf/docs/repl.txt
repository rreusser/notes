
{{alias}}( order, M, N, A, LDA, TAU, strideTAU, WORK, strideWORK )
    Reduces a complex M-by-N (M <= N) upper trapezoidal matrix to upper triangular form via the unitary RZ factorization (blocked driver).

    Parameters
    ----------
        order: string
            Row-major (C-style) or column-major (Fortran-style) order.

        M: integer
            Number of rows of `A`.

        N: integer
            Number of columns of `A` (must satisfy `N >= M`).

        A: Complex128Array
            Input/output matrix.

        LDA: integer
            Leading dimension of `A`.

        TAU: Complex128Array
            Output array of scalar factors of the elementary reflectors (length `M`).

        strideTAU: integer
            Stride length for `TAU`.

        WORK: Complex128Array
            Workspace array.

        strideWORK: integer
            Stride length for `WORK`.

    Returns
    -------
    out: integer
        Status code (`0` = success).


{{alias}}.ndarray( M, N, A, strideA1, strideA2, offsetA, TAU, strideTAU, offsetTAU, WORK, strideWORK, offsetWORK )
    Reduces a complex M-by-N (M <= N) upper trapezoidal matrix to upper triangular form via the unitary RZ factorization (blocked driver), using alternative indexing semantics.

    Parameters
    ----------
        M: integer
            Number of rows of `A`.

        N: integer
            Number of columns of `A`.

        A: Complex128Array
            Input/output matrix.

        strideA1: integer
            Stride of the first dimension of `A` (in complex elements).

        strideA2: integer
            Stride of the second dimension of `A` (in complex elements).

        offsetA: integer
            Starting index for `A`.

        TAU: Complex128Array
            Output array of scalar factors of the elementary reflectors.

        strideTAU: integer
            Stride length for `TAU` (in complex elements).

        offsetTAU: integer
            Starting index for `TAU`.

        WORK: Complex128Array
            Workspace array.

        strideWORK: integer
            Stride length for `WORK` (in complex elements).

        offsetWORK: integer
            Starting index for `WORK`.

    Returns
    -------
    out: integer
        Status code (`0` = success).
