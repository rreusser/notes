{{alias}}( order, jobz, uplo, N, kd, AB, LDAB, w, strideW, Z, LDZ, WORK, strideWORK )
    Computes all eigenvalues and optionally eigenvectors of a real symmetric
    band matrix.

    Parameters
    ----------
    order: string
        Row-major (C-style) or column-major (Fortran-style) order.

    jobz: string
        `jobz`.

    uplo: string
        specifies whether the upper or lower triangular part is referenced.

    N: integer
        number of columns.

    kd: number
        `kd`.

    AB: Float64Array
        `AB`.

    LDAB: integer
        leading dimension of `AB`.

    w: Float64Array
        `w`.

    strideW: integer
        stride length for `W`.

    Z: Float64Array
        `Z`.

    LDZ: integer
        leading dimension of `Z`.

    WORK: Float64Array
        `WORK`.

    strideWORK: integer
        stride length for `WORK`.
