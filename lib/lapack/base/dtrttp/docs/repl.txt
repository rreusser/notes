{{alias}}( order, uplo, N, A, LDA, AP )
    Copies a triangular matrix from full format (TR) to standard packed
    format (TP).

    Parameters
    ----------
    order: string
        Row-major (C-style) or column-major (Fortran-style) order.

    uplo: string
        specifies whether the upper or lower triangular part is referenced.

    N: integer
        number of columns.

    A: Float64Array
        `A`.

    LDA: integer
        leading dimension of `A`.

    AP: Float64Array
        `AP`.

    Returns
    -------
    out: Float64Array
