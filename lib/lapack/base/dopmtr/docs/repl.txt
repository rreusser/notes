{{alias}}( side, uplo, trans, M, N, AP, TAU, C, LDC, WORK )
    Overwrites a general matrix with a transformation from the orthogonal
    matrix Q returned by dsptrd.

    Parameters
    ----------
    side: string
        specifies the side of the operation.

    uplo: string
        specifies whether the upper or lower triangular part is referenced.

    trans: string
        specifies whether the matrix should be transposed.

    M: integer
        number of rows.

    N: integer
        number of columns.

    AP: Float64Array
        `AP`.

    TAU: Float64Array
        `TAU`.

    C: Float64Array
        `C`.

    LDC: integer
        leading dimension of `C`.

    WORK: Float64Array
        `WORK`.
