
{{alias}}( order, M, N, A, LDA, TAU, strideTAU, WORK, strideWORK )
    Computes a QR factorization of a real M-by-N matrix A = Q * R with
    non-negative diagonal elements of R, using a blocked algorithm.

    Parameters
    ----------
        order: string
            Row-major (C-style) or column-major (Fortran-style) order.

        M: integer
            number of rows.

        N: integer
            number of columns.

        A: Float64Array
            input matrix.

        LDA: integer
            leading dimension of `A`.

        TAU: Float64Array
            output array of scalar factors.

        strideTAU: integer
            stride length for `TAU`.

        WORK: Float64Array
            workspace array.

        strideWORK: integer
            stride length for `WORK`.

    Returns
    -------
    out: integer
        status code (0 on success).

    Examples
    --------
    > var A = new {{alias:@stdlib/array/float64}}( [ 2.0, 1.0, 3.0, 1.0, 4.0, 2.0, 3.0, 2.0, 5.0 ] );
    > var TAU = new {{alias:@stdlib/array/float64}}( 3 );
    > var WORK = new {{alias:@stdlib/array/float64}}( 96 );
    > {{alias}}( 'column-major', 3, 3, A, 3, TAU, 1, WORK, 1 )
    0


{{alias}}.ndarray( M, N, A, strideA1, strideA2, offsetA, TAU, strideTAU, offsetTAU, WORK, strideWORK, offsetWORK )
    Computes a QR factorization of a real M-by-N matrix A = Q * R with
    non-negative diagonal elements of R, using alternative indexing semantics.

    Parameters
    ----------
        M: integer
            number of rows.

        N: integer
            number of columns.

        A: Float64Array
            input matrix.

        strideA1: integer
            stride of dimension 1 of `A`.

        strideA2: integer
            stride of dimension 2 of `A`.

        offsetA: integer
            starting index for `A`.

        TAU: Float64Array
            output array of scalar factors.

        strideTAU: integer
            stride length for `TAU`.

        offsetTAU: integer
            starting index for `TAU`.

        WORK: Float64Array
            workspace array.

        strideWORK: integer
            stride length for `WORK`.

        offsetWORK: integer
            starting index for `WORK`.

    Returns
    -------
    out: integer
        status code (0 on success).

    Examples
    --------
    > var A = new {{alias:@stdlib/array/float64}}( [ 2.0, 1.0, 3.0, 1.0, 4.0, 2.0, 3.0, 2.0, 5.0 ] );
    > var TAU = new {{alias:@stdlib/array/float64}}( 3 );
    > var WORK = new {{alias:@stdlib/array/float64}}( 96 );
    > {{alias}}.ndarray( 3, 3, A, 1, 3, 0, TAU, 1, 0, WORK, 1, 0 )
    0
