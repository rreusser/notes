    Estimates the reciprocal of the condition number (in the 1-norm) of a
    real.

    Parameters
    ----------
    uplo: string
        specifies whether the upper or lower triangular part is referenced.

    N: integer
        number of columns.

    A: Float64Array
        `A`.

    LDA: integer

        `IPIV`.

    strideIPIV: integer

    WORK: Float64Array

    strideWORK: integer
        stride length for `WORK`.

    IWORK: Int32Array
