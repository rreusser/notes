{{alias}}( order, N, nrhs, A, LDA, IPIV, strideIPIV, B, LDB )
    Computes the solution to a real system of linear equations `A * X = B`
    where `A` is an N-by-N matrix.

    Parameters
    ----------
    order: string
        Row-major (C-style) or column-major (Fortran-style) order.

    N: integer
        Order of matrix A (number of rows and columns).

    nrhs: integer
        Number of right-hand side columns.

    A: Float64Array
        Input/output N-by-N matrix. On exit, L and U factors.

    LDA: integer
        Leading dimension of `A`.

    IPIV: Int32Array
        Output pivot indices (0-based), length N.

    strideIPIV: integer
        Stride length for `IPIV`.

    B: Float64Array
        Input/output N-by-NRHS matrix. On exit, the solution X.

    LDB: integer
        Leading dimension of `B`.

    Returns
    -------
    info: integer
        Status code. 0 if successful.

    Examples
    --------
    > var A = new {{alias:@stdlib/array/float64}}( [ 1.0, 3.0, 2.0, 4.0 ] );
    > var IPIV = new {{alias:@stdlib/array/int32}}( 2 );
    > var B = new {{alias:@stdlib/array/float64}}( [ 5.0, 11.0 ] );
    > var info = {{alias}}( 'column-major', 2, 1, A, 2, IPIV, 1, B, 2 )
    0
    > B
    <Float64Array>[ 1.0, 2.0 ]


{{alias}}.ndarray( N, nrhs, A, strideA1, strideA2, offsetA, IPIV, strideIPIV, offsetIPIV, B, strideB1, strideB2, offsetB )
    Computes the solution to a real system of linear equations `A * X = B`,
    using alternative indexing semantics.

    Parameters
    ----------
    N: integer
        Order of matrix A (number of rows and columns).

    nrhs: integer
        Number of right-hand side columns.

    A: Float64Array
        Input/output N-by-N matrix. On exit, L and U factors.

    strideA1: integer
        Stride of dimension 1 of `A`.

    strideA2: integer
        Stride of dimension 2 of `A`.

    offsetA: integer
        Starting index for `A`.

    IPIV: Int32Array
        Output pivot indices (0-based), length N.

    strideIPIV: integer
        Stride length for `IPIV`.

    offsetIPIV: integer
        Starting index for `IPIV`.

    B: Float64Array
        Input/output N-by-NRHS matrix. On exit, the solution X.

    strideB1: integer
        Stride of dimension 1 of `B`.

    strideB2: integer
        Stride of dimension 2 of `B`.

    offsetB: integer
        Starting index for `B`.

    Returns
    -------
    info: integer
        Status code. 0 if successful.

    Examples
    --------
    > var A = new {{alias:@stdlib/array/float64}}( [ 1.0, 3.0, 2.0, 4.0 ] );
    > var IPIV = new {{alias:@stdlib/array/int32}}( 2 );
    > var B = new {{alias:@stdlib/array/float64}}( [ 5.0, 11.0 ] );
    > var info = {{alias}}.ndarray( 2, 1, A, 1, 2, 0, IPIV, 1, 0, B, 1, 2, 0 )
    0
    > B
    <Float64Array>[ 1.0, 2.0 ]
