    Overwrites a real M-by-N matrix C with op(Q)*C or C*op(Q) using a compact WY representation.

    Parameters
    ----------
        order: string
            Row-major (C-style) or column-major (Fortran-style) order.

        side: string
            specifies the operation type.

        trans: string
            specifies the operation type.

        M: integer
            number of rows.

        N: integer
            number of columns.

        K: integer
            number of superdiagonals.

        nb: integer
            nb.

        V: Float64Array
            input matrix.

        LDV: integer
            leading dimension of `V`.

        T: Float64Array
            input matrix.

        LDT: integer
            leading dimension of `T`.

        C: Float64Array
            input matrix.

        LDC: integer
            leading dimension of `C`.

        WORK: Float64Array
            output array.

        strideWORK: integer
            stride length for `WORK`.
