    Forms the triangular factor T of a complex block reflector

    Parameters
    ----------
        order: string
            Row-major (C-style) or column-major (Fortran-style) order.

        direct: string
            specifies the operation type.

        storev: string
            specifies the operation type.

        N: integer
            number of columns.

        K: integer
            number of superdiagonals.

        V: Float64Array
            input matrix.

        LDV: integer
            leading dimension of `V`.

        TAU: Float64Array
            input array.

        strideTAU: integer
            stride length for `TAU`.

        T: Float64Array
            output matrix.

        LDT: integer
            leading dimension of `T`.
