{{alias}}( uplo, N, nrhs, A, strideA1, strideA2, offsetA, IPIV, strideIPIV, offsetIPIV, B, strideB1, strideB2, offsetB )
    Perform complex division, storing result in module-level cdR and cdI.

    Parameters
    ----------
    uplo: string
        specifies whether the upper or lower triangular part is referenced.

    N: integer
        number of columns.

    nrhs: integer
        number of right-hand sides.

    A: Float64Array
        `A`.

    strideA1: integer
        stride of dimension 1 of `A`.

    strideA2: integer
        stride of dimension 2 of `A`.

    offsetA: integer
        starting index for `A`.

    IPIV: Int32Array
        `IPIV`.

    strideIPIV: integer
        stride length for `IPIV`.

    offsetIPIV: integer
        starting index for `IPIV`.

    B: Float64Array
        `B`.

    strideB1: integer
        stride of dimension 1 of `B`.

    strideB2: integer
        stride of dimension 2 of `B`.

    offsetB: integer
        starting index for `B`.

    Returns
    -------
    out: Float64Array
        Result.
