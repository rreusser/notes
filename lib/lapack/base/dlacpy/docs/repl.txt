{{alias}}( order, uplo, M, N, A, LDA, B, LDB )
    Copies all or part of a matrix `A` to another matrix `B`.

    Parameters
    ----------
    order: string
        Row-major (C-style) or column-major (Fortran-style) order.

    uplo: string
        specifies whether the upper or lower triangular part is referenced.

    M: integer
        number of rows.

    N: integer
        number of columns.

    A: Float64Array
        `A`.

    LDA: integer
        leading dimension of `A`.

    B: Float64Array
        `B`.

    LDB: integer
        leading dimension of `B`.

    Returns
    -------
    out: Float64Array
        Result.
