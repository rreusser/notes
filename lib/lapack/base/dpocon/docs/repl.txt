{{alias}}( order, uplo, N, A, LDA, anorm, rcond, WORK, strideWORK, IWORK, strideIWORK )
    Estimates the reciprocal of the condition number of a symmetric
    positive.

    Parameters
    ----------
    order: string
        Row-major (C-style) or column-major (Fortran-style) order.

    uplo: string
        specifies whether the upper or lower triangular part is referenced.

    N: integer
        number of columns.

    A: Float64Array
        `A`.

    LDA: integer
        leading dimension of `A`.

    anorm: number
        `anorm`.

    rcond: number
        `rcond`.

    WORK: Float64Array
        `WORK`.

    strideWORK: integer
        stride length for `WORK`.

    IWORK: Int32Array
        `IWORK`.

    strideIWORK: integer
        stride length for `IWORK`.

    Returns
    -------
    out: Float64Array
        Result.
