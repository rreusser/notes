    Factorize a panel of a complex symmetric matrix using Aasen's algorithm.

    Parameters
    ----------
        order: string
            Row-major (C-style) or column-major (Fortran-style) order.

        uplo: string
            specifies the operation type.

        j1: integer
            j1.

        M: integer
            number of rows.

        nb: integer
            nb.

        A: Float64Array
            input matrix.

        LDA: integer
            leading dimension of `A`.

        IPIV: Int32Array
            input array.

        strideIPIV: integer
            stride length for `IPIV`.

        offsetIPIV: integer
            starting index for `IPIV`.

        H: Float64Array
            input matrix.

        LDH: integer
            leading dimension of `H`.

        WORK: Float64Array
            output array.

        strideWORK: integer
            stride length for `WORK`.

    Returns

    out: None
