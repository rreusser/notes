{{alias}}( order, uplo, N, nrhs, A, LDA, B, LDB )
    Computes the solution to a real system of linear equations A*X = B.

    Parameters
    ----------
    order: string
        Row-major (C-style) or column-major (Fortran-style) order.

    uplo: string
        specifies whether the upper or lower triangular part is referenced.

    N: integer
        number of columns.

    nrhs: integer
        number of right-hand sides.

    A: Float64Array
        `A`.

    LDA: integer
        leading dimension of `A`.

    B: Float64Array
        `B`.

    LDB: integer
        leading dimension of `B`.

    Returns
    -------
    out: Float64Array
        Result.

    using alternative indexing semantics.

    Parameters
    ----------
    uplo: string
        specifies whether the upper or lower triangular part is referenced.

    N: integer
        number of columns.

    nrhs: integer
        number of right-hand sides.

    A: Float64Array
        `A`.

    strideA1: integer
        stride of dimension 1 of `A`.

    strideA2: integer
        stride of dimension 2 of `A`.

    offsetA: integer
        starting index for `A`.

    B: Float64Array
        `B`.

    strideB1: integer
        stride of dimension 1 of `B`.

    strideB2: integer
        stride of dimension 2 of `B`.
