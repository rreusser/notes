{{alias}}( order, transr, uplo, N, A, LDA, ARF )
    Copies a triangular matrix from standard full format (TR) to Rectangular
    Full Packed format (RFP).

    Parameters
    ----------
    order: string
        Row-major (C-style) or column-major (Fortran-style) order.

    transr: string
        `transr`.

    uplo: string
        specifies whether the upper or lower triangular part is referenced.

    N: integer
        number of columns.

    A: Float64Array
        `A`.

    LDA: integer
        leading dimension of `A`.

    ARF: Float64Array
        `ARF`.
