    Parameters
    ----------
        order: string
            Row-major (C-style) or column-major (Fortran-style) order.

        side: string
            specifies the operation type.

        eigsrc: string
            specifies the operation type.

        initv: string
            specifies the operation type.

        SELECT: Float64Array
            input array.

        strideSELECT: integer
            stride length for `SELECT`.

        N: integer
            number of columns.

        H: Float64Array
            input matrix.

        LDH: integer
            leading dimension of `H`.

        WR: Float64Array
            input array.

        strideWR: integer
            stride length for `WR`.

        WI: Float64Array
            input array.

        strideWI: integer
            stride length for `WI`.

        VL: Float64Array
            input matrix.

        LDVL: integer
            leading dimension of `VL`.

        VR: Float64Array
            input matrix.

        LDVR: integer
            leading dimension of `VR`.

        mm: integer
            mm.

        M: integer
            number of rows.

        WORK: Float64Array
            input array.

        strideWORK: integer
            stride length for `WORK`.

        IFAILL: Int32Array
            input array.

            output array.

        strideIFAILR: integer
