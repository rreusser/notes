
{{alias}}( order, uplo, N, A, LDA, e, strideE, IPIV, strideIPIV, offsetIPIV, anorm, rcond, WORK, strideWORK )
    Estimates the reciprocal of the 1-norm condition number of a complex Hermitian matrix using the factorization computed by zhetrf_rk

    Parameters
    ----------
        order: string
            Row-major (C-style) or column-major (Fortran-style) order.

        uplo: string
            specifies the operation type.

        N: integer
            number of columns.

        A: Float64Array
            input matrix.

        LDA: integer
            leading dimension of `A`.

        e: Float64Array
            input array.

        strideE: integer
            stride length for `e`.

        IPIV: Int32Array
            input array.

        strideIPIV: integer
            stride length for `IPIV`.

        offsetIPIV: integer
            starting index for `IPIV`.

        anorm: number
            anorm.

        rcond: number
            rcond.

        WORK: Float64Array
            output array.

        strideWORK: integer
            stride length for `WORK`.

    Returns
    -------
    out: integer
        Result.
