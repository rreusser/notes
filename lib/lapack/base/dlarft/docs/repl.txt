    Forms the triangular factor T of a real block reflector H of order N.

    Parameters
    ----------
    order: string
        Row-major (C-style) or column-major (Fortran-style) order.

    direct: string
        `direct`.

    storev: string

    N: integer
        number of columns.

    K: integer
        inner dimension.

    V: Float64Array
        `V`.

    LDV: integer
        leading dimension of `V`.

    TAU: Float64Array
        `TAU`.

    strideTAU: integer
        stride length for `TAU`.

    T: Float64Array
        `T`.

    LDT: integer
        leading dimension of `T`.
