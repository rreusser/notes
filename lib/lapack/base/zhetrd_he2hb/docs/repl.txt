
{{alias}}( order, uplo, N, kd, A, LDA, AB, LDAB, TAU, strideTAU, WORK, strideWORK )
    Reduce a complex Hermitian matrix A to complex Hermitian band-diagonal form AB by a unitary similarity transformation: Q^H * A * Q = AB.

    Parameters
    ----------
        order: string
            Row-major (C-style) or column-major (Fortran-style) order.

        uplo: string
            specifies the operation type.

        N: integer
            number of columns.

        kd: integer
            kd.

        A: Float64Array
            input matrix.

        LDA: integer
            leading dimension of `A`.

        AB: Float64Array
            input matrix.

        LDAB: integer
            leading dimension of `AB`.

        TAU: Float64Array
            input array.

        strideTAU: integer
            stride length for `TAU`.

        WORK: Float64Array
            output array.

        strideWORK: integer
            stride length for `WORK`.

    Returns
    -------
    out: integer
        Result.
