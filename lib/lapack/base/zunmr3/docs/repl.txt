    multiplies a general matrix by the unitary matrix Q from an RZ factorization

    Parameters
    ----------
        order: string
            Row-major (C-style) or column-major (Fortran-style) order.

        side: string
            specifies the operation type.

        trans: string
            specifies the operation type.

        M: integer
            number of rows.

        N: integer
            number of columns.

        K: integer
            number of superdiagonals.

        l: integer
            l.

        A: Float64Array
            input matrix.

        LDA: integer
            leading dimension of `A`.

        TAU: Float64Array
            input array.

        strideTAU: integer
            stride length for `TAU`.

        C: Float64Array
            input matrix.

        LDC: integer
            leading dimension of `C`.

        WORK: Float64Array
            output array.

        strideWORK: integer
            stride length for `WORK`.
