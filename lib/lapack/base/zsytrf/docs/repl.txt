{{alias}}( uplo, N, A, strideA1, strideA2, offsetA, IPIV, strideIPIV, offsetIPIV )
    Compute the factorization of a complex symmetric matrix using
    Bunch-Kaufman diagonal pivoting.

    Parameters
    ----------
    uplo: string
        specifies whether the upper or lower triangular part is referenced.

    N: integer
        number of columns.

    A: Float64Array
        `A`.

    strideA1: integer
        stride of dimension 1 of `A`.

    strideA2: integer
        stride of dimension 2 of `A`.

    offsetA: integer
        starting index for `A`.

    IPIV: Int32Array
        `IPIV`.

    strideIPIV: integer
        stride length for `IPIV`.

    offsetIPIV: integer
        starting index for `IPIV`.
