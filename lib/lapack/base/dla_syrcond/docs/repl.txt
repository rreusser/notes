    Parameters
    ----------
        order: string
            Row-major (C-style) or column-major (Fortran-style) order.

        uplo: string
            specifies the operation type.

        N: integer
            number of columns.

        A: Float64Array
            input matrix.

        LDA: integer
            leading dimension of `A`.

        AF: Float64Array
            input matrix.

        LDAF: integer
            leading dimension of `AF`.

        IPIV: Int32Array
            input array.

        strideIPIV: integer
            stride length for `IPIV`.

        offsetIPIV: integer
            starting index for `IPIV`.

        cmode: integer
            cmode.

        c: Float64Array
            input array.

        strideC: integer
            stride length for `c`.

        WORK: Float64Array
            input array.

        strideWORK: integer
            stride length for `WORK`.

        IWORK: Int32Array
            output array.

        strideIWORK: integer
            stride length for `IWORK`.

        offsetIWORK: integer
            starting index for `IWORK`.
