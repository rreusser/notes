
{{alias}}( order, M, N, mb1, nb1, nb2, A, LDA, T, LDT, WORK, strideWORK )
    Computes a column-blocked QR factorization of a complex M-by-N matrix using TSQR followed by Householder reconstruction.

    Parameters
    ----------
        order: string
            Row-major (C-style) or column-major (Fortran-style) order.

        M: integer
            number of rows.

        N: integer
            number of columns.

        mb1: integer
            mb1.

        nb1: integer
            nb1.

        nb2: integer
            nb2.

        A: Float64Array
            input matrix.

        LDA: integer
            leading dimension of `A`.

        T: Float64Array
            input matrix.

        LDT: integer
            leading dimension of `T`.

        WORK: Float64Array
            output array.

        strideWORK: integer
            stride length for `WORK`.
