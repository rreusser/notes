{{alias}}( order, uplo, N, kd, nrhs, AB, LDAB, B, LDB )
    Solves a complex Hermitian positive definite banded system of equations.

    Parameters
    ----------
    order: string
        Row-major (C-style) or column-major (Fortran-style) order.

    uplo: string
        specifies whether the upper or lower triangular part is referenced.

    N: integer
        number of columns.

    kd: number
        `kd`.

    nrhs: integer
        number of right-hand sides.

    AB: Float64Array
        `AB`.

    LDAB: integer
        leading dimension of `AB`.

    B: Float64Array
        `B`.

    LDB: integer
        leading dimension of `B`.

    -------
    out: Float64Array
