    Apply a complex elementary reflector H to a complex M-by-N matrix C.

    Parameters
    ----------
    order: string
        Row-major (C-style) or column-major (Fortran-style) order.

    side: string
        specifies the side of the operation.

    M: integer
        number of rows.

    N: integer
        number of columns.

    v: Float64Array
        `v`.

    strideV: integer
        stride length for `V`.

    tau: number
        `tau`.

    offsetTau: integer
        starting index for `Tau`.

    C: Float64Array
        `C`.

    LDC: integer
        leading dimension of `C`.

    WORK: Float64Array
        `WORK`.

    strideWORK: integer
        stride length for `WORK`.
