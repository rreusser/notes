{{alias}}( order, jobz, uplo, N, AP, w, Z, LDZ, WORK )
    Computes all eigenvalues and, optionally, eigenvectors of a real
    symmetric matrix in packed storage.

    Parameters
    ----------
    order: string
        Row-major (C-style) or column-major (Fortran-style) order.

    jobz: string
        `jobz`.

    uplo: string
        specifies whether the upper or lower triangular part is referenced.

    N: integer
        number of columns.

    AP: Float64Array
        `AP`.

    w: Float64Array
        `w`.

    Z: Float64Array
        `Z`.

    LDZ: integer
        leading dimension of `Z`.

    WORK: Float64Array
        `WORK`.
