{{alias}}( order, uplo, N, nb, A, LDA, e, strideE, TAU, strideTAU, W, LDW )
    Reduces NB rows and columns of a complex Hermitian matrix A to
    Hermitian.

    Parameters
    ----------
    order: string
        Row-major (C-style) or column-major (Fortran-style) order.

    uplo: string
        specifies whether the upper or lower triangular part is referenced.

    N: integer
        number of columns.

    nb: integer
        `nb`.

    A: Float64Array
        `A`.

    LDA: integer
        leading dimension of `A`.

    e: Float64Array
        `e`.

    strideE: integer
        stride length for `E`.

    TAU: Float64Array
        `TAU`.

    strideTAU: integer
        stride length for `TAU`.

    W: Float64Array
        `W`.

    LDW: integer
        leading dimension of `W`.

    Returns

    out: Float64Array
