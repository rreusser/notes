{{alias}}( order, M, N, K, A, LDA, TAU, strideTAU, WORK, strideWORK )
    Generate an M-by-N real orthogonal matrix Q with orthonormal columns.

    Parameters
    ----------
    order: string
        Row-major (C-style) or column-major (Fortran-style) order.

    M: integer
        number of rows.

    N: integer
        number of columns.

    K: integer
        inner dimension.

    A: Float64Array
        `A`.

    LDA: integer
        leading dimension of `A`.

    TAU: Float64Array
        `TAU`.

    strideTAU: integer
        stride length for `TAU`.

    WORK: Float64Array
        `WORK`.

    strideWORK: integer
        stride length for `WORK`.
